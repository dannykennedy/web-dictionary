How to use this dictionary

  • The search bar looks for both Bininj Kunwok and English terms as well as dialect variations.
  • The dictionary is currently weighted towards Kunwinjku as spoken at Kunbarlanja, so if you're learning a different dialect, keep an eye on the 'other languages' section within certain entries.
  • It's still a work in progress, and in all questions of correctness and appropriate use you should defer to native speakers.

This dictionary was developed by the Bininj Kunwok Regional Language Centre for Charles Darwin University's Kunwinjku Language course.
